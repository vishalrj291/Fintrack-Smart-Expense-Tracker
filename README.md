Fintrack-Smart Expense Tracker

A modern, responsive, and visually stunning expense management application built with React, Vite, Tailwind CSS, and Framer Motion.

Designed with a premium fintech-inspired UI, this application helps users monitor spending habits, track budgets, visualize financial data, and gain insights into their expenses through interactive dashboards and analytics.

Installation

Clone the repository:

git clone https://github.com/your-username/smart-expense-tracker.git

Navigate to project directory:

cd smart-expense-tracker

Install dependencies:

npm install

Run development server:

npm run dev

Open:

http://localhost:5173
Build for Production
npm run build

Preview production build:

npm run preview
